Información sobre Doxygen

Doxygen admite el uso de la sintaxis de Markdown (Consulta aquí los comandos básicos).

Así como el uso de sintaxis HTML. Esto es especialmente útil para aquellos aspectos que Markdown no permite modificar, como el color. Aunque es buena idea evitarlo en la medida de lo posible, por dos motivos:

  1. Hace el código poco LEGIBLE
  2. La idea detrás de Markdown es separar aspecto y contenido. Y es una gran idea.

La forma más correcta de modificar el aspecto debería ser mediante un documento .css configurado en el archivo Doxyfile.in. Si te interesa, puedes echarle un vistazo a la documentación o a temas customizados como este. Aunque el tema por defecto nos parece perfecto, CLÁSICO. Así puedes centrarte en el contenido.

Por último, lo hemos visto en clase y hay suficientes ejemplos en el código fuente proporcionado, pero puedes consultar la mayor parte de la sintaxis de Doxygen para comentar código en su documentación.

Ejecutables

Negativo:

Calcula el negativo de una imagen

negativo <FichImagenOriginal> <FichImagenDestino>

Parameters
<FichImagenOriginal>Imagen PGM a la que se va a calcular
<FichImagenDestino>Imagen PGM resultado de calcular el negativo

Subimagen:

Calcula la subimagen de una imagen

subimagen <FichImagenOriginal> <FichImagenDestino> <numeroFilas> <numeroColumnas> <altura> <ancho>

Parameters
<FichImagenOriginal>Imagen PGM a la que se va a calcular subimagen
<FichImagenDestino>Imagen PGM resultado de calcular la subimagen
<numeroFilas>La fila desde donde se empieza a coger pixeles
<numeroColumnas>La columna desde donde se empieza a coger pixeles
<altura>La altura que va a tener la imagen
<ancho>El ancho que va a tener la imagen

zoom:

Calcula el zoom de una imagen

zoom <FichImagenOriginal> <FichImagenDestino> <lado> <columna> <lado>

Parameters
<FichImagenOriginal>Imagen PGM a la que se va a calcular el zoom
<FichImagenDestino>Imagen PGM resultado de calcular el zoom
<lado>La fila desde donde se empieza a coger pixeles para zoom
<columna>La columna desde donde se empieza a coger pixeles para zoom
<lado>La altura y anchura de la imagen ladoxlado

icono:

Calcula el icono de una imagen

icono <FichImagenOriginal> <FichImagenDestino> <factor>

Parameters
<FichImagenOriginal>Imagen PGM a la que se va a calcular el icono
<FichImagenDestino>Imagen PGM resultado de calcular el icono
<factor>El factor que se le aplica al icono

contraste:

Calcula la constraste de una imagen

contraste <FichImagenOriginal> <FichImagenDestino> <in1> <in2> <out1> <out2>

Parameters
<FichImagenOriginal>Imagen PGM a la que se va a calcular el contraste
<FichImagenDestino>Imagen PGM resultado de calcular el contraste
<in1>El comienzo del intervalo de inicio
<in2>El final del intervalo de inicio
<out1>EL principio del intervalo de salida
<out2>El final del intervalo de salda
